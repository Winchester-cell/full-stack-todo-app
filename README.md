## Project Goal

The main goal of this project was to **practice Next.js and SSR concepts**, as well as **backend development and security**. Even though the project is simple, it covers most of the essential web development concepts.

---

## Technologies Used

**Frontend:** Next.js, Tailwind CSS, React Query, Zustand
**Backend:** Next.js API Routes, MongoDB, JWT, Nodemailer
**Special Features:**

* Avatar upload via Uploadcare
* Secure token management
* Password recovery with OTP
* Middleware to protect routes

---

## Key Implementations & Learning

* Developed a secure and professional REST API
* Implemented user authentication and refresh token logic
* Designed database models with virtual fields for optimized storage
* Managed state and API requests efficiently using React Query and Zustand
* Built a modular and scalable project structure

## How to Run Locally

```bash
# Clone the repository
git clone <your-repo-link>

# Install dependencies
npm install

# Run the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
